# stripe-utils

This package contains shared utilities and helper methods for the [Stripe](https://www.stripe.com/) payment provider.
It includes common logic (e.g., payment methods filtering) that is specific to Stripe integration package and can be reused across different integration layers.

## Running unit tests

Run `nx test stripe-utils` to execute the unit tests via [Jest](https://jestjs.io).

## Running lint

Run `nx lint stripe-utils` to execute the lint via [ESLint](https://eslint.org/).
